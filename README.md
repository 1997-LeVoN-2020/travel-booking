# Travel Booking Management System

Система управления бронированиями отелей на базе Django.

## Описание проекта

Этот проект представляет собой веб-приложение для управления бронированиями в сети отелей. Система позволяет:

- Управлять информацией об отелях
- Управлять типами номеров и их доступностью
- Отслеживать бронирования
- Работать с тарифами и ценами
- Синхронизировать данные между системами
- Управлять пользователями и их правами

## Установка

### Требования

- Python 3.13+
- pip (менеджер пакетов Python)
- Git

### Шаги установки

1. **Клонируем репозиторий**
   ```bash
   git clone https://github.com/yourusername/travel-booking.git
   cd travel
   ```

2. **Создаём виртуальное окружение**
   ```bash
   python -m venv venv
   ```

3. **Активируем виртуальное окружение**
   - На Windows:
     ```bash
     venv\Scripts\activate
     ```
   - На macOS/Linux:
     ```bash
     source venv/bin/activate
     ```

4. **Устанавливаем зависимости**
   ```bash
   pip install -r requirements.txt
   ```

5. **Применяем миграции базы данных**
   ```bash
   python manage.py migrate
   ```

6. **Создаём суперпользователя**
   ```bash
   python manage.py createsuperuser
   ```

7. **Запускаем приложение**
   ```bash
   python manage.py runserver
   ```

Приложение будет доступно по адресу `http://127.0.0.1:8000/`

## Приложения

### hotels
Управление информацией об отелях, типами номеров и контактными лицами.

### bookings
Управление бронированиями и их статусами.

### dashboard
Панель управления с аналитикой и отчётами.

### rates
Управление тарифами и ценами на номера.

### sync
Синхронизация данных с внешними системами.

### users
Управление пользователями системы.

## Использование

### Вход в админ-панель
1. Перейдите на `http://127.0.0.1:8000/admin/`
2. Введите учетные данные суперпользователя

### Добавление отеля
1. Перейдите в админ-панель
2. В разделе "Hotels" нажмите "Add Hotel"
3. Заполните необходимую информацию
